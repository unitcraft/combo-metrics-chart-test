# План работ — Combo Metrics Chart

## Цель (из тестового задания)
Воспроизвести график, который принимает **4 time-series последовательности** и рисует их
разными типами: **area / spline / line / column (bar)**. Внимательно повторить **стиль и
поведение** эталона, выложить **GitHub-репозиторий** с инструкцией инициализации четырьмя
последовательностями данных.

Эталон: <https://cleanshot.com/share/cBDRxJWM> (анимированный GIF; статичный кадр сохранён в
[`docs/reference.jpg`](reference.jpg)).

## Поведение, воспроизведённое из эталона
- Четыре серии разных типов на одном поле (area / spline / line / column).
- Отдельная ось Y на каждую серию (единицы `$` / `%` / число не конфликтуют).
- Общий тултип — белая скруглённая карточка с датой-заголовком и цветным маркером + значением.
- Crosshair (пунктир), halo на активной точке, квадратные маркеры на линии.
- Розовый фон поля, плавная анимация появления.

## Решения
**Колонки CPA.** В эталоне синие колонки короткие (на крупном `$`-масштабе). Реализовано
через `axisMax: 18` для оси CPA — колонки прижаты к основанию, совпадение с эталоном 1:1.
